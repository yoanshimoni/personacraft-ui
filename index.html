LinkedIn AI Assistant

This Chrome extension helps you write warm, personalized outreach messages on LinkedIn using AI. Just visit a profile and the assistant will generate suggestions for connection requests or direct messages, tailored to your goals.

Features

  • 🧠 Personalized message suggestions
  • ⚡ Fast AI-powered response
  • 🔐 No data stored or shared

How to Use

  1. Install the extension from the Chrome Web Store
  2. Open the extension and save your intent - what you are looking for
  3. Visit any LinkedIn profile
  4. Open the “Add Note” modal or LinkedIn Messaging
  5. Your AI assistant will suggest messages automatically
